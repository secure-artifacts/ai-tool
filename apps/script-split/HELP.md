## ✂️ 脚本拆分

### AI 自动拆分
1. 输入完整脚本
2. 选择拆分模式：
   - **按场景**：根据场景转换拆分
   - **按镜头**：根据镜头描述拆分
   - **按时间**：按固定时长拆分
   - **按段落**：按段落拆分
3. 点击「AI 拆分」

### 手动拆分
- 使用分隔符标记拆分点
- 支持的分隔符：`---`、`###`、空行
- 手动添加/删除分镜行

### 拆分规则

| 规则 | 说明 |
|------|------|
| 场景转换 | 场景变化时拆分 |
| 镜头切换 | 镜头运动变化时拆分 |
| 角色切换 | 说话角色变化时拆分 |
| 时间跳跃 | 时间明显跳跃时拆分 |

## 📋 分镜编辑

### 分镜表字段

| 字段 | 说明 |
|------|------|
| 镜号 | 分镜序号 |
| 画面描述 | 画面内容描述 |
| 台词/旁白 | 对话或旁白内容 |

### 行操作
- **添加行**：在当前行下方插入新行
- **删除行**：删除当前行
- **上移/下移**：调整分镜顺序
- **合并行**：合并相邻分镜
- **拆分行**：将一行拆为多行

### 批量编辑
- 多选行批量操作
- 批量设置景别/运镜
- 批量调整时长

## ❓ 常见问题

### Q: AI 拆分不准确怎么办？
**A:** 
1. 确保脚本格式清晰
2. 使用分隔符标记明显的拆分点
3. 手动调整不准确的拆分

### Q: 如何统一修改景别？
**A:** 使用多选功能，选中多行后批量设置景别。

### Q: 时长估算不准确？
**A:** 在设置中调整语速参数，根据实际播报速度调整。

### Q: 可以导入已有的分镜表吗？
**A:** 可以，支持导入 Excel 格式的分镜表。
