## ✍️ 提示词创作

### 提示词结构
推荐的提示词结构：

```
[角色设定]
你是一名...

[任务描述]
请帮我...

[输出要求]
- 格式：...
- 风格：...
- 长度：...

[示例/参考]
例如：...
```

### 最佳实践

| 技巧 | 说明 |
|------|------|
| 明确角色 | 告诉 AI 扮演什么角色 |
| 具体任务 | 清晰描述要完成的任务 |
| 输出格式 | 指定期望的输出格式 |
| 给出示例 | 提供参考示例效果更好 |
| 设定约束 | 说明不要做什么 |

### 变量使用
使用 `{{变量名}}` 格式定义可替换变量：

```
请以{{风格}}的风格，写一篇关于{{主题}}的文章。
```

## 🔧 变量系统

### 变量类型

| 类型 | 语法 | 说明 |
|------|------|------|
| 文本 | `{{name}}` | 单行文本输入 |
| 多行 | `{{name:textarea}}` | 多行文本输入 |
| 选择 | `{{name:select:选项1,选项2}}` | 下拉选择 |
| 数字 | `{{name:number}}` | 数字输入 |

### 示例
```
请以{{风格:select:正式,轻松,幽默}}的风格，
写一篇{{字数:number}}字左右的{{类型:select:新闻,评论,故事}}，
主题是：{{主题:textarea}}。
```

## ❓ 常见问题

### Q: 如何让提示词更有效？
**A:** 
1. 角色+任务+格式+示例四要素
2. 使用具体、明确的描述
3. 分步骤描述复杂任务

### Q: 预设可以分享给其他人吗？
**A:** 可以，使用导出功能将预设导出为 JSON 文件分享。

### Q: 如何备份我的预设？
**A:** 定期使用「全部导出」功能备份所有预设。

### Q: 变量不生效怎么办？
**A:** 检查变量语法是否正确，确保使用 `{{变量名}}` 格式。
