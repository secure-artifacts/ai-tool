## 🚀 快速开始

### 第一步：输入文案
1. 在左侧输入框粘贴需要检测的文案
2. 或上传文本文件

### 第二步：开始查重
1. 点击「开始检测」按钮
2. 系统会分析文案的重复情况
3. 显示重复率和具体位置

### 第三步：优化去重
1. 查看重复内容高亮标记
2. 使用AI一键改写重复部分
3. 导出优化后的文案

### 底部操作区
- **开始检测**：启动查重分析
- **一键优化**：AI 自动改写重复内容
- **导出结果**：导出报告和优化后文案

## 🔎 查重功能

### 检测模式

| 模式 | 说明 |
|------|------|
| 标准检测 | 检测句子级别的重复 |
| 严格检测 | 检测短语级别的重复 |
| 语义检测 | 检测意思相近的内容 |

### 重复率指标

| 重复率 | 评级 | 建议 |
|--------|------|------|
| 0-15% | ✅ 优秀 | 原创性高，可直接使用 |
| 15-30% | ⚠️ 一般 | 建议适当修改 |
| 30-50% | 🔶 较高 | 需要较多改写 |
| >50% | ❌ 过高 | 建议重写 |

### 结果解读
- **绿色**：原创内容
- **黄色**：轻微相似
- **红色**：高度重复

## ✨ 去重优化

### AI 改写
1. 选择需要改写的重复段落
2. 点击「AI 改写」按钮
3. 选择改写风格
4. 应用改写结果

### 改写风格

| 风格 | 说明 |
|------|------|
| 保持原意 | 换种表达，意思不变 |
| 精简压缩 | 删除冗余，更加简洁 |
| 扩展丰富 | 增加细节，更加丰富 |
| 口语化 | 更加通俗易懂 |
| 书面化 | 更加正式专业 |

### 手动编辑
- 直接在结果区编辑文本
- 实时更新重复率
- 支持撤销重做

## ❓ 常见问题

### Q: 检测准确率如何？
**A:** 采用先进的 AI 语义分析，准确率较高。但建议配合人工复核。

### Q: 支持多大的文件？
**A:** 单次检测建议不超过 5 万字。更长文档建议分段检测。

### Q: 改写后意思会变吗？
**A:** AI 改写尽量保持原意，但建议检查确认。

### Q: 可以排除引用内容吗？
**A:** 可以，使用「排除引用」功能标记引用内容。

## 💡 使用技巧

1. **分段检测**：长文分段检测效率更高
2. **多次优化**：重复率高的内容多次改写
3. **人工复核**：AI 改写后人工确认意思
4. **保留报告**：保留检测报告作为原创证明
