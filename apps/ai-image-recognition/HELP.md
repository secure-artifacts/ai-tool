## 🚀 快速开始

### 第一步：上传图片
1. 点击上传区域或拖拽图片到界面
2. 支持批量上传多张图片
3. 支持 JPG、PNG、GIF、WebP 格式

### 第二步：配置分析
1. 选择分析模式（自动识别 / 自定义提示词）
2. 设置输出语言
3. 可选：启用纯回复模式

### 第三步：开始分析
1. 点击「开始分析」按钮
2. AI 会逐张分析每张图片
3. 结果显示在图片下方

### 顶部操作区
- **上传按钮**：点击上传图片文件

## 📤 上传图片

### 支持的格式
| 格式 | 说明 |
|------|------|
| JPG/JPEG | 通用照片格式 |
| PNG | 支持透明背景 |
| GIF | 静态和动态图 |
| WebP | 现代压缩格式 |

### 上传方式
1. **点击上传**：点击上传区域选择文件
2. **拖拽上传**：直接拖拽图片到界面
3. **剪贴板粘贴**：Ctrl/⌘ + V 粘贴截图
4. **网络图片**：输入图片 URL

### 批量上传
- 支持一次选择多张图片
- 无数量限制（建议单次不超过 50 张）
- 自动生成缩略图预览

## 🤖 AI 分析

### 分析模式

| 模式 | 说明 |
|------|------|
| 自动识别 | AI 自动描述图片内容 |
| 物体识别 | 识别图片中的物体 |
| 文字提取 | OCR 提取图片中的文字 |
| 场景分析 | 分析图片场景和氛围 |
| 自定义 | 使用自定义提示词 |

### 自定义提示词
在提示词框中输入你的分析指令，例如：
- "请描述这张图片的主要内容"
- "识别图片中的品牌和产品"
- "分析这张图片的色彩搭配"
- "提取图片中的所有文字"

### 纯回复模式
启用后，AI 只返回分析结果，不添加额外说明。适合需要结构化输出的场景。

## ❓ 常见问题

### Q: 图片分析失败怎么办？
**A:** 
1. 检查网络连接
2. 确认 API 密钥已配置
3. 尝试重新上传图片
4. 点击"重试"按钮

### Q: 支持多大的图片？
**A:** 单张图片建议不超过 20MB。较大图片会自动压缩。

### Q: 如何提高识别准确率？
**A:** 
1. 使用清晰、高分辨率的图片
2. 避免模糊、过曝或过暗的图片
3. 使用更具体的自定义提示词

### Q: 分析结果可以编辑吗？
**A:** 是的，点击结果可以直接编辑内容。
