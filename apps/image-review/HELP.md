# 📸 图片审核工具 (Image Review)

专门用于审核 AI 生成的图片，提供结构化的反馈流程，支持中英文双语反馈验证。
**专为「中国审核员 → 外国制作团队」的跨语言、跨文化协作场景优化。**

---

## 🎯 核心功能

### 1. 审核状态
| 状态 | 快捷键 | 说明 |
|-----|-------|------|
| ✅ 合格 | `1` | 图片通过审核 |
| ❌ 不合格 | `2` | 图片未通过 |
| ✏️ 需要修改 | `3` | 需要特定修改 |
| 🚫 放弃 | `4` | 完全放弃此图 |

### 2. 双栏反馈结构
每个反馈项包含两个部分：
- **❌ 问题描述 (Problem)**：描述问题是什么
- **💡 改进建议 (Suggestion)**：建议如何解决

### 3. 问题严重程度
| 级别 | 说明 |
|-----|------|
| 🔴 严重 (Critical) | 必须修复的关键问题 |
| 🟡 中等 (Major) | 需要修复的重要问题 |
| 🔵 轻微 (Minor) | 可选修复的小问题 |
| 💡 建议 (Suggestion) | 优化建议，非必须 |

---

## 🌐 跨文化沟通优化

### 翻译验证 + 语气设置
- **中文反馈**：直接输入中文建议
- **英文翻译**：自动翻译为英文
- **回译确认**：英文回译为中文验证准确性
- **语气级别**：
  - **中性 (Neutral)**：客观陈述问题
  - **建议式 (Suggestive)**：温和友好的建议方式 ✨ 推荐
  - **协作式 (Collaborative)**：强调团队合作

### 💡 语气示例对比

| 中文原文 | 中性 | 建议式 ✨ | 协作式 |
|---------|------|---------|--------|
| 这里颜色不对 | The color is wrong here. | It might be helpful to adjust the color here. | Let's take a look at the color together - perhaps we could adjust it? |

## 📚 预设常用语（预翻译语料库）

**一张参考图胜过十句翻译完美的英文！**

针对高频美术问题，提供**人工校对过的专业英文表述**：

| 分类 | 示例标签 |
|-----|---------|
| 人体结构 | 手指数量错误、人体比例失调、面部不自然 |
| 贴图材质 | 贴图模糊、穿模问题、贴图接缝、金属感不足 |
| 光影效果 | 阴影不正确、光线过硬、光影平淡 |
| 颜色配色 | 颜色过于鲜艳、颜色不匹配、对比度问题 |
| 构图布局 | 裁切不当、构图不平衡、焦点不清晰 |
| 技术问题 | 分辨率不足、压缩伪影、噪点过多 |
| 风格氛围 | 风格不统一、氛围不对、细节不足 |

**使用方法**：
1. 点击「常用语」按钮
2. 选择分类或搜索关键词
3. 点击短语自动填入（已包含预翻译的英文）

## 🎨 辅助工具

### 📎 参考图贴入
与其写 "Make it more metallic"，不如直接贴一张你觉得合格的金属图！

1. 点击反馈项中的「参考图」按钮
2. 选择一张参考图片
3. 图片会自动附加到反馈中
4. 执行清单视图会显示「📎 Reference (Like this)」

### 🎨 取色器 + Hex 代码
颜色问题？不要写 "深一点的红色"，直接给 Hex 代码！

1. 点击「颜色」按钮
2. 用取色器选择精准颜色
3. 系统自动生成 Hex 代码（如 #FF0033）
4. 代码是全世界通用的语言，绝对不会有歧义！

## 🖌️ 图片标注

在图片上绘制标注来指出问题位置：
- **□ 矩形**：框选问题区域
- **○ 圆形**：圈出问题点
- **→ 箭头**：指向问题位置
- **✏️ 画笔**：自由绘制
- **T 文字**：添加文字说明

## 📋 执行清单视图 (Execution Checklist)

**专为外国团队设计的任务导向视图！**

与审核视图（按图片）不同，执行清单视图**按任务**聚合反馈：

- ❌ **不合格** - 必须修复的问题
- ✏️ **需要修改** - 需要调整的内容
- 💡 **建议优化** - 可选的改进建议

### 优势
1. 团队成员看到的是**待办事项**，而非图片列表
2. 相似问题自动聚合，避免重复描述
3. 显示受影响的图片缩略图
4. 显示参考图和推荐颜色代码
5. 一键复制整个清单发送给团队

### 切换方式
工具栏右侧有「审核」和「执行清单」两个按钮，点击切换视图。

## 📤 报告导出

### PDF 报告
点击「导出报告」→「打印/下载 PDF」，生成专业的审核报告：
- 项目信息摘要
- 审核状态统计
- 问题严重程度分布
- 每张图片的详细反馈

### 文本报告（Google Docs）
1. 点击「导出报告」→「下载文本」
2. 打开 Google Docs
3. 粘贴内容即可

### 复制到剪贴板
快速复制报告内容，直接粘贴到邮件或聊天工具中发送。

## 📝 工作流程示例

### 1. 准备工作
1. 点击「项目信息」填写项目名称、审核人等信息
2. 设置翻译语气（推荐：建议式）
3. 导入待审核的图片（拖拽/粘贴/选择文件）

### 2. 逐图审核
1. 点击图片进入详细视图
2. 设置审核状态（合格/不合格/需修改/放弃）
3. 添加反馈项：
   - 使用「常用语」快速添加预翻译反馈
   - 或自定义输入
4. 如需更精准表达：
   - 贴入参考图（一图胜千言）
   - 使用取色器指定颜色

### 3. 翻译与导出
1. 点击「翻译所有反馈」翻译为英文
2. 检查回译确认翻译准确
3. 切换到「执行清单」视图预览团队视角
4. 点击「导出报告」或复制清单发送给制作团队
